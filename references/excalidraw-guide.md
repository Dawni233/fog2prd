# Excalidraw 图表生成指南

本文档定义了原型设计中 Excalidraw 图表的生成规则和模板。

---

## 1. 图表类型

### 1.1 三种核心图表

| 图表类型 | 用途 | 包含内容 |
|----------|------|----------|
| 业务流程图 | 展示业务流程 | 角色、流程节点、判断分支、流向 |
| 页面线框图 | 展示页面布局 | 页面结构、组件位置、字段布局 |
| 交互流程图 | 展示页面跳转 | 页面节点、操作触发、跳转关系 |

---

## 2. 业务流程图生成

### 2.1 流程图元素

| 元素 | 形状 | 颜色 | 说明 |
|------|------|------|------|
| 开始/结束 | 椭圆 | 浅绿 #b2f2bb | 流程起点和终点 |
| 流程步骤 | 矩形 | 浅蓝 #a5d8ff | 具体操作步骤 |
| 判断/分支 | 菱形 | 浅黄 #fff3bf | 条件判断 |
| 数据/文档 | 平行四边形 | 浅紫 #d0bfff | 数据输入输出 |
| 连接线 | 箭头 | 蓝色 #3b82f6 | 流程方向 |
| 泳道 | 矩形区域 | 半透明背景 | 区分角色 |

### 2.4 流程图生成规则

1. **识别流程步骤**：从功能需求中提取操作步骤
2. **确定判断节点**：识别条件分支
3. **划分角色**：涉及多角色时使用泳道
4. **绘制流程**：按从上到下或从左到右顺序

## 5. 图表生成规则

### 5.1 业务流程图生成

**输入**：功能需求中的操作流程

**生成步骤**：
1. 提取流程步骤
2. 识别开始和结束节点
3. 确定判断分支点
4. 按顺序连接节点
5. 如有多角色，添加泳道

**输出**：Excalidraw 格式的业务流程图

### 5.2 页面线框图生成

**输入**：页面清单和页面元素定义

**生成步骤**：
1. 根据页面类型选择布局模板
2. 按元素清单放置组件
3. 添加字段标签和占位符
4. 标注操作按钮

**输出**：Excalidraw 格式的页面线框图

### 5.3 交互流程图生成

**输入**：页面清单和页面关系

**生成步骤**：
1. 列出所有页面节点
2. 根据页面关系连接节点
3. 标注触发操作
4. 区分页面跳转和弹窗

**输出**：Excalidraw 格式的交互流程图

## 6. 调用 excalidraw-diagram Skill

### 6.1 业务流程图调用示例

```
生成业务流程图：
[请假审批流程]

参与者：员工、部门经理、HR

流程步骤：
1. 员工提交请假申请
2. 系统发送通知给部门经理
3. 部门经理审批
4. 如果通过，流转到HR备案
5. 如果驳回，返回员工修改
6. HR备案完成，流程结束
```

### 6.2 页面线框图调用示例

```
生成页面线框图：

页面类型：列表页
页面名称：用户管理

布局结构：
- Header: Logo + 系统名称 + 用户头像
- Sidebar: 导航菜单
- Main: 页面标题 + 操作按钮 + 搜索区 + 表格 + 分页

表格字段：用户名、姓名、部门、状态、创建时间、操作

操作按钮：新增、导出、编辑、删除
```

### 6.3 交互流程图调用示例

```
生成交互流程图：

页面列表：
- 登录页
- 用户管理（列表）
- 新增用户（弹窗）
- 编辑用户（弹窗）
- 用户详情（页面）

交互关系：
- 登录页 → 用户管理（登录成功）
- 用户管理 → 新增用户（点击新增）
- 用户管理 → 编辑用户（点击编辑）
- 用户管理 → 用户详情（点击查看）
- 新增用户 → 用户管理（保存成功，刷新）
- 编辑用户 → 用户管理（保存成功，刷新）
```
