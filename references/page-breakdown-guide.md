# 页面拆解指南

本文档定义了从功能需求中识别和拆解页面的规则和方法。

---

## 1. 页面拆解流程

### 1.1 拆解步骤

```
功能需求列表
      │
      ▼
┌─────────────────┐
│ 1. 识别页面类型  │  按功能分类确定页面类型
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 2. 确定页面关系  │  页面间跳转和导航
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 3. 定义页面元素  │  布局、组件、字段
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 4. 设计交互流程  │  操作、反馈、跳转
└────────┬────────┘
         │
         ▼
    页面清单
```

## 2. 页面类型识别

### 2.1 页面类型对照表

| 功能类型 | 页面类型 | 说明 |
|----------|----------|------|
| 数据录入（新增/编辑） | 表单页 | 填写表单提交数据 |
| 数据查询（列表/搜索） | 列表页 | 数据表格展示和筛选 |
| 数据详情 | 详情页 | 单条数据完整信息 |
| 审批流程 | 审批页 | 审批操作和意见填写 |
| 数据统计 | 仪表盘/报表页 | 图表和数据汇总 |
| 系统配置 | 设置页 | 参数配置和开关 |
| 用户登录/注册 | 登录页/注册页 | 身份认证 |

### 2.2 功能到页面映射

**示例：用户管理模块**

| 功能 | 页面 | 页面类型 |
|------|------|----------|
| 用户列表查询 | 用户管理页 | 列表页 |
| 添加用户 | 用户添加弹窗/页 | 表单页 |
| 编辑用户 | 用户编辑弹窗/页 | 表单页 |
| 查看用户详情 | 用户详情页 | 详情页 |
| 删除用户 | 确认弹窗 | 弹窗 |

## 4. 页面关系设计

### 4.1 页面关系类型

| 关系类型 | 说明 | 示例 |
|----------|------|------|
| 导航跳转 | 侧边栏菜单导航 | 首页→用户管理 |
| 列表到详情 | 查看详细内容 | 用户列表→用户详情 |
| 列表到表单 | 新增/编辑操作 | 用户列表→添加用户 |
| 详情到表单 | 编辑操作 | 用户详情→编辑用户 |
| 流程跳转 | 审批流程流转 | 待审批→审批页→已审批 |
| 弹窗打开 | 轻量操作 | 确认删除弹窗 |

### 4.2 页面关系图

```
                    ┌─────────┐
                    │  首页   │
                    └────┬────┘
                         │
         ┌───────────────┼───────────────┐
         │               │               │
         ▼               ▼               ▼
   ┌──────────┐    ┌──────────┐    ┌──────────┐
   │ 用户管理  │    │ 角色管理  │    │ 系统设置  │
   └────┬─────┘    └──────────┘    └──────────┘
        │
        ├──────────┬──────────┐
        │          │          │
        ▼          ▼          ▼
   ┌────────┐ ┌────────┐ ┌────────┐
   │添加用户│ │编辑用户│ │用户详情│
   │(弹窗)  │ │(弹窗)  │ │(页面)  │
   └────────┘ └────────┘ └────┬───┘
                             │
                             ▼
                        ┌────────┐
                        │编辑用户│
                        │(弹窗)  │
                        └────────┘
```

### 5.1 页面元素清单模板

```markdown
### [页面名称]

**页面类型**：列表页/表单页/详情页

**入口位置**：[从哪里进入]

**页面元素**：

| 元素名称 | 元素类型 | 说明 |
|----------|----------|------|
| 页面标题 | 文本 | "用户管理" |
| 新增按钮 | 按钮 | 主操作，打开新增表单 |
| 搜索框 | 输入框 | 关键词搜索 |
| 状态筛选 | 下拉选择 | 全部/启用/禁用 |
| 数据表格 | 表格 | 显示用户列表 |
| 操作列 | 操作按钮 | 编辑、删除、查看详情 |
| 分页 | 分页组件 | 数据分页 |

**字段列表**（列表页/表格）：

| 序号 | 字段名 | 宽度 | 是否排序 | 说明 |
|------|--------|------|----------|------|
| 1 | 用户名 | 150px | 是 | - |
| 2 | 姓名 | 100px | 否 | - |
| 3 | 部门 | 150px | 否 | - |
| 4 | 状态 | 80px | 否 | 标签样式 |
| 5 | 操作 | 150px | 否 | 固定右侧 |

**表单字段**（表单页）：

| 序号 | 字段名 | 类型 | 必填 | 说明 |
|------|--------|------|------|------|
| 1 | 用户名 | 输入框 | 是 | - |
| 2 | 姓名 | 输入框 | 是 | - |
| 3 | 部门 | 树选择 | 是 | - |
| 4 | 角色 | 穿梭框 | 是 | 多选 |

**交互说明**：

| 操作 | 触发条件 | 行为 |
|------|----------|------|
| 点击新增 | 点击按钮 | 弹出新增表单 |
| 点击编辑 | 点击表格操作列 | 弹出编辑表单（预填数据） |
| 点击删除 | 点击表格操作列 | 弹出确认弹窗 |
| 点击搜索 | 点击按钮或回车 | 刷新表格数据 |
| 点击重置 | 点击按钮 | 清空搜索条件 |
```

## 6. 完整示例

### 用户管理模块页面拆解

**功能需求**：
- F001 用户列表查询
- F002 用户添加
- F003 用户编辑
- F004 用户删除
- F005 用户详情查看
- F006 用户状态切换

**页面拆解结果**：

```markdown
## 页面清单

| 序号 | 页面名称 | 页面类型 | 入口 | 关联功能 |
|------|----------|----------|------|----------|
| P01 | 用户管理 | 列表页 | 系统管理菜单 | F001, F006 |
| P02 | 用户添加 | 表单弹窗 | P01新增按钮 | F002 |
| P03 | 用户编辑 | 表单弹窗 | P01编辑按钮 | F003 |
| P04 | 删除确认 | 确认弹窗 | P01删除按钮 | F004 |
| P05 | 用户详情 | 详情页 | P01查看按钮 | F005 |

### P01 用户管理页

**页面类型**：列表页

**入口位置**：系统管理 > 用户管理

**页面元素**：

| 元素名称 | 元素类型 | 说明 |
|----------|----------|------|
| 页面标题 | 文本 | "用户管理" |
| 新增按钮 | 主按钮 | 打开用户添加弹窗 |
| 导入按钮 | 按钮 | 批量导入用户 |
| 导出按钮 | 按钮 | 导出用户列表 |
| 用户名搜索 | 输入框 | 模糊搜索 |
| 部门筛选 | 树选择 | 按部门筛选 |
| 状态筛选 | 下拉选择 | 全部/启用/禁用 |
| 搜索按钮 | 按钮 | 执行搜索 |
| 重置按钮 | 按钮 | 清空条件 |
| 用户表格 | 表格 | 用户数据列表 |
| 分页组件 | 分页 | 数据分页 |

**表格字段**：

| 序号 | 字段名 | 宽度 | 排序 | 说明 |
|------|--------|------|------|------|
| 1 | 用户名 | 120px | ✓ | - |
| 2 | 姓名 | 100px | - | - |
| 3 | 部门 | 150px | - | 显示部门路径 |
| 4 | 手机号 | 120px | - | 脱敏显示 |
| 5 | 角色 | 150px | - | 多角色逗号分隔 |
| 6 | 状态 | 80px | - | 启用(绿)/禁用(灰) |
| 7 | 创建时间 | 150px | ✓ | YYYY-MM-DD HH:mm |
| 8 | 操作 | 180px | - | 固定右侧 |

**操作列按钮**：

| 按钮 | 说明 | 权限 |
|------|------|------|
| 编辑 | 打开编辑弹窗 | user:edit |
| 删除 | 打开删除确认 | user:delete |
| 重置密码 | 重置为默认密码 | user:reset |
| 查看详情 | 跳转详情页 | user:view |

### P02 用户添加弹窗

**页面类型**：表单弹窗

**入口位置**：P01 新增按钮

**弹窗尺寸**：600px × auto

**表单字段**：

| 序号 | 字段名 | 类型 | 必填 | 校验规则 |
|------|--------|------|------|----------|
| 1 | 用户名 | 输入框 | ✓ | 4-20字符，唯一 |
| 2 | 姓名 | 输入框 | ✓ | 2-20字符 |
| 3 | 手机号 | 输入框 | ✓ | 11位手机号 |
| 4 | 邮箱 | 输入框 | - | 邮箱格式 |
| 5 | 部门 | 树选择 | ✓ | 必选 |
| 6 | 角色 | 穿梭框 | ✓ | 至少1个 |
| 7 | 状态 | 开关 | ✓ | 默认启用 |

**操作按钮**：

| 按钮 | 说明 |
|------|------|
| 取消 | 关闭弹窗 |
| 确定 | 校验并保存 |

**交互说明**：
- 用户名失焦时进行唯一性校验
- 手机号失焦时校验格式和唯一性
- 部门选择器支持搜索
- 提交成功后关闭弹窗并刷新列表
```
